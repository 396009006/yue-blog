# yue-blog

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### 项目介绍
```
1.基于 vue-cli
2.后端基于 spring + spring mvc + mybatis
```
